# Quick-Add

Google カレンダーのタイムラインで右クリックすると、よく使うイベントタイトルを即座に入力できる Chrome 拡張機能です。初期メニューを自由にカスタマイズし、イベント作成の手間を減らします。

## 機能

- Google カレンダーのタイムライン上で右クリックするとカスタムメニューを表示
- メニューの選択で Google カレンダーのイベント作成ダイアログが開き、タイトルが初期入力された状態に
- メニュー項目は拡張機能のポップアップから追加・編集・削除が可能
- 初期メニューは以下を同梱
  - 【開発】
  - 【レビュー】
  - 【CS】
  - 【MTG】
  - 【業務改善】
  - 【採用】
  - 【Help】
  - 【調査】
  - 【その他】

## 使い方

1. リポジトリをダウンロードし、Chrome で `chrome://extensions` を開きます。
2. 右上の「デベロッパーモード」をオンにし、「パッケージ化されていない拡張機能を読み込む」をクリックします。
3. 本ディレクトリを選択すると拡張機能が追加されます。
4. Google カレンダーを開き、タイムライン上で右クリックするとカスタムメニューが表示されます。
5. 拡張機能のアイコンをクリックするとメニュー管理画面が開き、項目の追加・編集・削除ができます。

## 開発メモ

- 初回インストール時にデフォルトメニューが自動登録されます。削除したい場合はポップアップから削除してください。
- Google カレンダー側の UI 変更でイベント作成ダイアログの構造が変わった場合、タイトル入力欄が見つからず動作しない可能性があります。その際は `content/contentScript.js` のセレクタを更新してください。
